# Archive Layout

The `archive/` tree preserves historical material without leaving it on the
public repo landing surface.

Key buckets:

- `archive/snapshots/`: older pilot-output directories moved off the root
- `archive/extracted/`: extracted snapshot directories moved off the root
- `archive/misc/`: notes, logs, and one-off drop artifacts
- `archive/zip/`: preserved zip deliverables and bundle drops
- `archive/dirs/`: archived directory-form bundles and historical work packages

Archive placement is organizational, not dismissive. Material moved here is
kept for provenance and replay unless it is explicitly deleted in a later
commit.
