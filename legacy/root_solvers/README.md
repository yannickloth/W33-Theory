# Legacy Root Solvers

These scripts were historically kept at the repository root as one-off or
parallel standalone solver experiments.

They were moved under `legacy/root_solvers/` to make the public repo entry
surface narrower and more professional:

- active theorem work now lives primarily in `exploration/`, `tests/`, `docs/`,
  `tools/`, and `scripts/`
- these files are preserved for historical continuity and ad hoc replay
- they are not the recommended starting point for new readers or contributors

If you are trying to understand or extend the current promoted stack, start
with:

- `README.md`
- `docs/index.html`
- `docs/march_2026_frontier_note.md`
- `docs/REPO_STRUCTURE.md`
