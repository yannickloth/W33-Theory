# 120-block (edgepair) Bose–Mesner multiplication table

Basis A0..A4 where A0=I and A1..A4 are the 4 orbitals with valencies (2,27,36,54).

- A1·A1 = 2A0 + 1A1
- A1·A2 = 1A4
- A1·A3 = 2A3
- A1·A4 = 2A2 + 1A4
- A2·A2 = 27A0 + 10A2 + 6A3 + 4A4
- A2·A3 = 8A2 + 9A3 + 8A4
- A2·A4 = 27A1 + 8A2 + 12A3 + 14A4
- A3·A3 = 36A0 + 36A1 + 12A2 + 6A3 + 12A4
- A3·A4 = 16A2 + 18A3 + 16A4
- A4·A4 = 54A0 + 27A1 + 28A2 + 24A3 + 22A4
